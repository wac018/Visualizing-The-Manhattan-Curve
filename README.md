# Visualizing-The-Manhattan-Curve

## How to Run

The parameters `num_points`,`t_1`, `t_2`, `max_depth`, and `show_status_updates` can be configured in the `run_program.m` file.

### Parameters

1. `num_points` : controls the number of points you want to approximate on the Manhattan Curve.
2. `t_1` and `t_2` : The parameters that control the shape of the strictly convex sets Omega_1 and Omega_2.
3. `max_depth` : controls the number of matrices that are calculated.
4. `show_status_updates` : set to` true` if you would like status updates on the progress of matrix calculations. Setting to `false` will supress these update messages.


To run, type `run_program` in the MATLAB console and hit enter to run the program.
